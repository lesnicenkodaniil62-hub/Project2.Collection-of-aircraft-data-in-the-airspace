## Инструкция по настройке линтеров для данного проекта

Создайте файл с названием ".flake8" и в нём введите следующие
```
[flake8]
max-line-length = 119
ignore = E402, E251, E203
exclude = .git, .venv, __pycache__, venv

```
Вводите в файл pyproject.toml, если там не прописано, то пропишите следующие.
```
[tool.black]
# Максимальная длина строки
line-length = 119
# Файлы, которые не нужно форматировать
exclude = '''
(
  /(
      \.eggs         # Исключить несколько общих каталогов
    | \.git          # в корне проекта
    | \.hg
    | \.mypy_cache
    | \.tox
    | \.venv
    | dist
  )/
  | foo.py           # Также отдельно исключить файл с именем foo.py
                     # в корне проекта
)
'''


[tool.isort]
# максимальная длина строки
line_length = 119


[tool.mypy]
disallow_untyped_defs = true
warn_return_any = true
ignore_missing_imports = true
exclude = "venv"

```
И все зависимости что есть на данный момент будут прописаны.
